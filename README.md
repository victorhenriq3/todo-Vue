# todo-Vue
Simple To Do project in Vuejs

# Getting started

# Install
- [Node 12x](https://nodejs.org/en/)
- [Vue CLI](https://cli.vuejs.org/)

# Clone Repository and Open Directory
```shell
git clone https://github.com/victorhenriq3/todo-Vue.git
cd todo-Vue
```
# Install dependeces
```shell
npm install
```
# Run
```shell
npm run serve
```
